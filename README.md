# suiton_confirm_form_pack
## concrete5 5.7系用確認画面付メールフォーム

- jQueryを使った確認画面表示 
- jQuery.varidation.engineでフロントでのバリデーション
- 管理者/送信者毎の件名設定
- 管理者/送信者毎の自動返信メール文設定
- メール本文の最初に記載する「〜様」的なテキストの設定

※基本はformブロックの拡張

## 使い方

ブロックに「Suiton Confirm Form」を追加 
各種項目設定（既存のフォームブロックとほぼ同じ流れ） 
テンプレート「confirm」を選択
